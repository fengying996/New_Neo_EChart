## 1. 页面定位
页面是一个单屏展示型可视化页面，核心目标是让三层旭日图成为绝对视觉中心，适合汇报、截图、打印和嵌入 PPT。

## 2. 视觉方向
- 风格关键词：专业、干净、可打印、图表主导
- 背景：纯白色
- 主文字色：深灰接近黑色
- 字体：微软雅黑优先，兼容中文系统办公环境
- 图表区域：大尺寸居中，四周保留呼吸感留白

## 3. 配色规范
### 3.1 熟练度基础色
| 熟练度 | 颜色 | 用途 |
|-------|------|------|
| `high` | `#2E7D32` | 高频精通，主强调色 |
| `medium` | `#F9A825` | 中频熟练，次强调色 |
| `low` | `#9E9E9E` | 低频试用，中性色 |

### 3.2 分层颜色策略
| 层级 | 颜色规则 |
|------|----------|
| 内层 `category_name` | 采用该分类下主导工具熟练度颜色的略深版本 |
| 中层 `tool_name` | 直接采用工具熟练度基础色 |
| 外层 `practice` | 采用对应工具基础色的略浅版本 |

## 4. 版式设计
| 区域 | 设计要求 |
|------|----------|
| 标题区 | 左上或居上，简短标题配副标题，不喧宾夺主 |
| 图表区 | 占页面主要面积，默认居中显示，保持 1:1 视觉比例 |
| 工具栏 | 使用 ECharts 内置工具栏，放置在右上角，保留保存图片功能 |
| 提示框 | 白底细描边，文字紧凑清晰，避免遮挡核心标签 |

## 5. 图表细节要求
- 三层圆环厚度均衡，避免最外层过窄导致文字无法识别
- 环间边界用白色描边，增强层次感
- 标签沿扇形径向排列，字号随层级递减
- 最内层字号最大，中层次之，外层最小
- 小角度扇区允许截断文字，但不隐藏主要信息
- 鼠标悬停时高亮祖先路径，便于理解上下级关系

## 6. 打印与导出要求
- 保持纯白背景，不使用深色底或复杂纹理
- 保存图片时使用白色背景和较高像素比
- 页面内容避免依赖网络字体，保证本地环境可稳定呈现
